# KubeQuest

**Think you know Kubernetes? Prove it.**  

KubeQuest is a fast-paced quiz game that challenges your K8s knowledge across 60+ questions, from basic Pod concepts to advanced scheduling and security scenarios.

## How to Play

Pick a difficulty, answer questions before time runs out, and try not to lose all your lives.  
Build **combos** for bonus points and aim for the S grade.

| Difficulty | Timer | Lives | Style |
|------------|-------|-------|-------|
| 🟢 **Recruit** | 15s | ❤️❤️❤️ | Core concepts, vocabulary |
| 🔵 **Engineer** | 12s | ❤️❤️🩶 | Commands, debugging, workloads |
| 🟣 **Architect** | 8s | ❤️🩶🩶 | Security, scheduling, real-world scenarios |

## Question Types

- **MCQ** : pick the right answer among four choices.
- **True / False** : quick-fire knowledge checks.
- **Ordering** : drag and drop steps in the correct sequence.
- **Type-in** : no hints, type the exact answer.

## Grading

Your performance is rated from **S** down to **D**, based on accuracy. Every correct answer earns points, and consecutive correct answers build a combo multiplier.  
Wrong answers cost a ❤️ : run out and it's game over.
